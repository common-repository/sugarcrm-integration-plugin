== Description ==

= Integrate SugarCRM to your Wordpress Admin Panel =

*SugarCRM Integration* is the best way to integrate your SugarCRM into the WordPress Admin Panel. The plugin was developed for SugarCRM community edition and it's without any hidden limitations!

= Tested versions: =
* Community Edition up to 6.3.1.

Feel free to contact us if you are having problems or ideas for new features on our german [plugin site](http://sugarcrm-integration.bitweise.net/). English comments are welcome and will be replied too!

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust settings to your SugarCRM page
4. Enjoy!
